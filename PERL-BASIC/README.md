# DAY 1 - Perl Basics and Logic

This directory contains Perl scripts covering fundamental concepts and solving various logical problems.

## Files

### `BASIC_PERL.pl`
Introduction to Perl syntax and data types.
- **Scalars**: Variables, string concatenation, arithmetic operations.
- **Arrays**: Creation, accessing elements, slicing, splicing, sorting.
- **Hashes**: Key-value pairs, accessing values, slicing.
- **Control Structures**: `if`, `elsif`, `else`, and user input handling (`STDIN`, `chomp`).
- **Operators**: String and numeric operators.

### `Condition_statement_Question2.pl`
**Problem: Multi-Stage Transaction Validator**
- Validates a list of transactions based on verification status.
- Counts accepted, rejected, and flagged transactions.
- Uses array of hash references.

### `Condition_statement_question.pl`
A collection of solutions to various problems using conditional statements and loops:
1. **Even/Odd Checker**: Determines if a number is even or odd.
2. **Department List**: Iterates through a list of departments.
3. **Countdown Script**: Uses a `while` loop for a countdown.
4. **Production Target Checker**: Compares production units against a target.
5. **Attendance Validator**: Categorizes attendance (Excellent, Satisfactory, Poor).
6. **Odd Number Printer**: Prints odd numbers, skipping multiples of 5.
7. **Order Processing**: Simulates processing statuses, stopping on failure.
8. **Age Validator**: Counts customers above 18.
9. **Compound Interest Calculator**: Calculates interest over years.

### `PERL_QUESTIONS_SOLUTION_DAY1.pl`
Comprehensive solutions for Day 1 questions, including:
- **Context**: String vs Numeric context.
- **Temperature Conversion**: Celsius to Fahrenheit with input validation.
- **Data Structures**: Hash and Array manipulation (slicing, splicing).
- **Loops**: `while`, `until`, `for`, `foreach`, `do-while`.
- **Algorithms**: Duplicate detection, Prime number check, GCD calculation.
- **String Manipulation**: Reversing words in a string.
