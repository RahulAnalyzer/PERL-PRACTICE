# JSON File Handling

This directory contains Perl scripts for working with JSON data.

## Files

### `CSV-To-JSON.pl`
Converts data from a CSV file into a structured JSON format.

### `jsonFile.pl`
Demonstrates how to read and parse a JSON file into a Perl data structure.

### `mergingJson.pl`
Shows how to combine or merge data from multiple JSON sources.

### `WriteJson.pl`
Illustrates how to encode a Perl hash or array into a JSON string and write it to a file.
