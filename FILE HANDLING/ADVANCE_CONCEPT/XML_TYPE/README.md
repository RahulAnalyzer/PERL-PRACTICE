# XML File Handling

This directory contains Perl scripts for processing XML files using various modules like `XML::LibXML` and `XML::Simple`.

## Files

### `reading_xmlfile.pl`
Basic script for reading and parsing an XML file to extract its content.

### `modifying_xml.pl`
Demonstrates how to programmatically change values or structure within an existing XML document.

### `studentXML.pl`
Specifically handles student-related data in XML format.

### `xmlTocsv.pl`
Flattens an XML structure and exports the data into a CSV file.

### `xmlTojson.pl`
Converts XML data into a JSON format for web or API compatibility.

### `Xml_type.pl`
General exploration of XML parsing and manipulation techniques.
